# Webtel Mobile Handoff

This folder is a separate mobile design workspace for the merged HR Pearls and iPay experience.

## What this scaffold includes

- A unified mobile shell for both products.
- A task-first navigation model: Home, Attendance, People, Pay, Requests, and More.
- Design tokens for spacing, color, typography, borders, and states.
- Simple screen switching so the developer can extend each section into real flows.

## Product direction from research

- HR Pearls is the employee self-service and HR operations surface.
- iPay is the payroll and employee-pay access surface.
- The merged app should keep HR tasks and payroll tasks separate in the IA, but visually consistent.

## Platform guidance applied

- Large tap targets and clear hierarchy.
- Bottom navigation for the primary paths.
- Calm layout density with strong contrast and accessible color states.
- iOS and Android friendly structure, so it can be turned into native screens later.

## Run it

Open `mobile-app/index.html` in a browser, or serve the folder with any static web server.
