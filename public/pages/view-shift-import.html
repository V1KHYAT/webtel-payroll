Assign Shift Through Import

Attention: Please follow the below mentioned rules else attendance might be calculated wrong.
  1. Do not left blank any mandatory fields in Import Excel file.
  2. Mandatory fields are Emp_Code, From_Date, To_Date and Shift_Code.
  3. Enter From_Date and To_date in excel sheet as 10-Jul-10 (DD-MMM-YY) or 10 Jul 2010 (DD MMM YYYY) Format (mandatory fields).
  4. If there is any wrong input in any cell, you need to correct it first and then re-import the whole excel sheet again or delete that row and re-import.
  5. Above is the excel sheet that will open once you click on Export.
  6. By default the file would get saved in XML. You need to save the file in XLS format.
  7. Check "Populate Last Shift" to populate last month shift.
  8. Check "Complete Month Format" to populate excel sheet in datewise format.
  9. If a user assigns more than one Shift in a month to an employee then be careful in inserting From_Date and To_Date.
